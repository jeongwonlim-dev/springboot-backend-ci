## Spring Boot Backend with CI/CD

This project is a simplified backend service built with Spring Boot.
It is intended to demonstrate backend development and CI/CD fundamentals.

This project is inspired by real-world systems I worked on during my
professional experience as a Full Stack Developer in Korea.

## Tech Stack
- Java
- Spring Boot
- Gradle / Maven
- GitHub Actions (CI)
- REST API

## Goals
- Build a clean and simple Spring Boot backend
- Apply CI automation for build and test
- Demonstrate production-oriented project structure
